# Student Management System

A backend REST API application built using Java, Spring Boot, Spring Data JPA, and MySQL to manage student records.  
This project demonstrates a clean layered architecture and basic CRUD operations used in real-world backend development.

>> 🚀 Features

- Create Student
- View All Students
- View Student by ID
- Update Student Details
- Delete Student
- RESTful API design
- Layered architecture (Controller → Service → Repository)
- MySQL database integration
- Spring Boot auto configuration

>> 🛠 Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Hibernate
- Maven
- IntelliJ IDEA
- Postman (API testing).
